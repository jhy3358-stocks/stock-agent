# 주식 에이전트팀 — 프로젝트 요구사항

## 1. 목적
매일 아침 국내/미국 주식 및 주요 지수에 대한 기술적 분석 리포트를 자동 생성하여 카카오톡으로 발송한다.

## 3. 리포트 내용
각 종목/지수별로 다음 정보를 포함한다:
- 현재가 및 등락률
- 이동평균선 (5일, 20일, 60일) 대비 위치
- RSI (14일 기준)
- 거래량 (전일 대비 증감)

## 4. 데이터 소스 (무료)
- **국내 종목/지수**: `pykrx` 라이브러리
- **미국 종목/지수**: `yfinance` 라이브러리
- (참고용, 우선순위 낮음) 뉴스 요약: 추후 확장 시 네이버 뉴스 검색 API 고려

## 5. 발송 방식
- **채널**: 카카오톡 "나에게 보내기" API (Kakao Developers, 무료/검수 불필요)
- **인증**: 최초 1회 OAuth 로그인으로 refresh token 발급 후 저장, 이후 자동 갱신
- **라이브러리**: `PyKakao` 또는 직접 REST API 호출

## 6. 실행 스케줄
- **시간**: 매일 오전 6~7시 (미국장 마감 후, KST 기준)
- **실행 환경**: GitHub Actions (cron 스케줄, PC 상시 구동 불필요)

## 7. 아키텍처

```
[GitHub Actions 스케줄 트리거 - 매일 06:00 KST]
        │
   ┌────┴────┬──────────┐
데이터수집   기술지표계산  (국내: pykrx / 해외: yfinance)
 에이전트    에이전트
        │
   리포트 텍스트 생성
        │
   카카오톡 "나에게 보내기" 발송
```

## 8. 개발 순서 (제안)
1. 카카오 "나에게 보내기" 인증 및 발송 스크립트 단독 완성 (refresh token 발급/저장 포함)
2. 국내 종목 데이터 수집 함수 (pykrx)
3. 해외 종목 데이터 수집 함수 (yfinance)
4. 지수 데이터 수집 함수
5. 기술적 지표 계산 모듈 (이동평균, RSI, 거래량 비교)
6. 리포트 텍스트 포맷팅
7. 전체 파이프라인 연결 및 로컬 테스트
8. GitHub Actions 워크플로우 작성 및 스케줄 등록 (secrets에 카카오 토큰 저장)

## 9. 향후 확장 아이디어 (미확정, 참고용)
- 뉴스/공시 요약 추가
- MACD, 볼린저밴드 등 추가 지표
- 매수/매도 의견 코멘트
